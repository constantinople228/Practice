# Practice
В рамках задания необходимо реализовать backend-сервис, который: 
- собирает информацию о версиях программного обеспечения из открытых 
интернет-источников (например, GitHub), 
- сохраняет полученные данные в базу данных, 
- предоставляет доступ к собранным данным через REST API с поддержкой 
пагинации.
